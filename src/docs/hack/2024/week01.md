# 从零开始打造一个编译器

## 介绍

通过构建一个编译器来了解编程语言的工作原理。使用低级汇编语言来了解计算机的运作方式。

演示一个最小但完整的编译器实现。将一个静态类型的语言编译成 x64 ELF 可执行文件。

内容包括：

- 简单的解释器。启动并运行您自己的编程语言。
- 字节码编译器。更接近底层机器的一步。
- x64 汇编与指令编码。了解机器的工作原理。
- 将字节码翻译成 x64 代码。连接虚拟机与真实机器。
- 生成二进制可执行文件。将所有组件整合在一起。
